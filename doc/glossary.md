# 用語集

| 用語 | 説明 | 英語名 (参考) |
| :--- | :--- | :--- |
| 最大コンテキスト長 | モデルが一度に処理できるコンテキストの最大量を示すトークン数。システムプロンプト、ユーザー入力、過去の会話履歴の合計がこの値を超える場合、コンテキストの圧縮が必要になる。 | Max Context Length |
| 会話履歴 | ユーザーとAI間のやり取り（ターン）の時系列リスト。 | Chat History |
| 枝刈り | 会話履歴が最大コンテキスト長を超える場合に、古い情報から削除してトークン数を上限内に収める処理。 | Pruning |
| システムプロンプト | 会話全体の前提条件や、AIに与える役割・応答スタイルを定義する特別な指示。通常、会話履歴の先頭に配置され、枝刈りの対象外となる。 | System Prompt |
| ターン | ユーザーの発言（`user`ロール）と、それに対するAIの応答（`assistant`ロール）から成る、会話の1往復分の単位。 | Turn |
