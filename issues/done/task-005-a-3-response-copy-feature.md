# 014: Task - LLM応答コピー機能の実装

## 概要

LLMが生成した応答メッセージのテキスト内容を、ユーザーがWebUIおよびCLIのREPLから簡単にクリップボードにコピーできるようにする機能を実装します。

## 関連仕様書

- [LLM応答コピー機能 仕様書](../doc/response_copy_feature_spec.md)

## 受け入れ基準 (Acceptance Criteria)

- **UI (Gradio)**
  - [ ] WebUIの各LLM応答メッセージの横に「コピー」ボタンが配置されている。
  - [ ] 「コピー」ボタンをクリックすると、その応答のテキスト内容がクリップボードにコピーされる。
  - [ ] コピー成功時にユーザーにフィードバックがある（例: 短い表示）。
- **CLI (REPL)**
  - [ ] REPLで `/copy <message_index>` コマンドが利用できる。
  - [ ] `/copy` コマンドで指定したインデックスのLLM応答がクリップボードにコピーされる。
  - [ ] コピー成功時にREPLにメッセージが表示される。
- **共通**
  - [ ] 新機能に関するユニットテストが追加され、すべてのテストがパスする。

## タスク分割 (Sub-tasks)

1.  **Chat Logic**: `chat_logic.py` に、指定されたインデックスのLLM応答メッセージを取得するメソッドを追加する。
2.  **Web UI**: `webui.py` を修正し、`gr.Chatbot` の表示をカスタマイズするか、JavaScriptを組み込むことで、各LLM応答の横に「コピー」ボタンを配置する。ボタンクリック時に、対応する応答テキストをクリップボードにコピーするロジックを実装する。
3.  **CLI**: `cli.py` のREPLに `/copy` コマンドのパーサーと、`chat_logic` からメッセージを取得し、クリップボードにコピーするロジックを実装する（`pyperclip` などのライブラリの導入を検討）。
4.  **Testing**: `tests/` ディレクトリ内の関連テストファイル（`test_chat_logic.py`, `test_webui.py`, `test_cli.py` 等）に、応答コピー機能の動作を検証するユニットテストを追加する。
5.  **Documentation**: `README.md` や関連ドキュメントを更新し、新しい応答コピー機能の使い方を説明する。
